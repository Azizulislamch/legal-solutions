## ⚖️ Justice - Effective Legal Solutions

A professional, responsive landing page for a modern Law Firm, designed to showcase legal expertise with a clean and authoritative UI. 
This project focuses on high-performance styling and interactive user experience.

## 🎨 Preview

### ✨ Features
- **Modern Hero Section:** A high-impact banner with a dark gradient overlay and integrated search functionality.
- **Interactive FAQ Section:** A custom-built accordion using Vanilla JavaScript for smooth open/close transitions.
- **Dynamic Practice Areas:** A clean 3-column grid layout showcasing various legal services with custom-styled action buttons.
- **Fully Responsive Design:** Optimized for all devices using a "Mobile First" strategy, ensuring a seamless experience from 320px to 1440px.
- **Themed Components:** Consistent branding using the font-garamond typeface and a professional #B49164 (Golden) color palette.
- **Professional Contact Form:** A user-friendly contact section with localized office information and working hours.

---

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for SEO and accessibility.
- **Tailwind CSS:** For utility-first, rapid UI development.
- **JavaScript (ES6+):** For interactive components like the FAQ accordion and navigation.
- **Font Awesome:** For crisp, scalable vector icons.
- **Google Fonts:** Specifically Cormorant Garamond for that classic legal aesthetic.
